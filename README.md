## Quick Start

1. Clone this repo:

  ```bash
  git clone https://github.com/Muhnyx/PSRG_website
  ```

1. Install Hugo (<https://gohugo.io/installation/>)
1. "hugo server" on the base and use the link in the terminal to access the site locally
